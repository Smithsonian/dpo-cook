### Configuration

Example configuration for MeshSmith in the `tools.json` configuration file:

"MeshSmith": {
    "executable": "C:\\Tools\\MeshSmith\\MeshSmith.exe",
    "version": "2019-09-12",
    "maxInstances": 3,
    "timeout": 600 // 10 minutes
}
