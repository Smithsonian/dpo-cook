### Configuration

Example configuration for Instant Meshes in the `tools.json` configuration file:

 ```json
"InstantMeshes": {
    "executable": "C:\\Tools\\Instant Meshes.exe",
    "version": "1",
    "maxInstances": 1,
    "timeout": 900
}
```
