### Configuration

Example configuration for RawTherapee in the `tools.json` configuration file:

```json
"RawTherapee": {
    "executable": "C:\\Program Files\\RawTherapee\\5.10\\rawtherapee-cli.exe",
    "version": "5.10",
    "maxInstances": 3,
    "timeout": 7200
}
```
