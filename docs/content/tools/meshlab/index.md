### Configuration

Example configuration for Meshlab in the `tools.json` configuration file:

```json
"Meshlab": {
    "executable": "C:\\Tools\\MeshlabMini\\meshlabserver.exe",
    "version": "v2019.5, 2019-05-02",
    "maxInstances": 3,
    "timeout": 1800
}
```
