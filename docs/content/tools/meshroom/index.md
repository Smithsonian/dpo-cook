### Configuration

Example configuration for Meshroom in the `tools.json` configuration file:

```json
"RealityCapture": {
    "executable": "C:\\Program Files\\Meshroom\\Meshroom-2021.1.0\\meshroom_batch.exe",
    "version": "2021.1.0",
    "maxInstances": 1,
    "timeout": 0 // never
}
```
