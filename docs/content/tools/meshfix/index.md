### Configuration

Example configuration for Meshfix in the `tools.json` configuration file:

```json
"Meshfix": {
    "executable": "C:\\Tools\\MeshFix.exe",
    "version": "v2.1",
    "maxInstances": 1,
    "timeout": 600
}
```
